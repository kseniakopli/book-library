# Лендинг nocturne (задача 69)

Самодостаточная страница: один `index.html` со встроенными стилями, без сборки.
Открывается двойным кликом, деплоится куда угодно.

## Что заполнить перед публикацией

1. **Форма листа ожидания.** В `index.html` два раза встречается
   `action="ВСТАВЬТЕ_ССЫЛКУ_ФОРМЫ"` — подставь адрес своей формы. Бесплатные
   варианты, бэкенд не нужен:
   - [Formspree](https://formspree.io) — регистрируешь форму, получаешь
     `https://formspree.io/f/xxxxxxx`, письма падают тебе на почту;
   - [Tally](https://tally.so) или Google Forms — если хочешь готовую таблицу
     подписчиков.
2. **Почта для связи.** Заменить `ВАША_ПОЧТА@example.com` (в блоке «Или просто напишите»).
3. **Скриншоты.** Положить в `screens/`: `1.png` (полка), `2.png` (страница книги
   в палитре), `3.png` (печатная карточка). Размер — ширина ~1200px достаточно.
4. **Скринкаст.** Записать короткое видео (полка → книга → «Начать вечер»),
   сохранить как `screencast.mp4` рядом с `index.html`. Если видео не будет —
   просто удали секцию «Вечер за минуту».
5. **OG-превью.** После деплоя заменить `<meta property="og:image">` на полный
   адрес картинки (например `https://nocturne.ru/screens/1.png`) — тогда ссылка
   в мессенджерах будет с картинкой.

## Как опубликовать

- **Netlify / Cloudflare Pages** — перетащить папку `landing/` в веб-интерфейс,
  получить адрес за минуту. Самый быстрый путь.
- **GitHub Pages** — включить Pages для репозитория и указать папку `landing/`
  (или скопировать её содержимое в ветку `gh-pages`).

## Домен

Проверить свободен ли `nocturne.*` — на любом регистраторе (reg.ru, namecheap).
Имя популярное, `nocturne.com` почти наверняка занят; смотри в сторону
`nocturne.app`, `nocturne.club`, `nocturne.books`, или с приставкой
(`nocturne-club`, `mynocturne`). Проверять лучше сразу и в зоне `.ru`.

## Заметка про git

Скриншоты (`screens/`) стоит закоммитить — они лёгкие. А вот `screencast.mp4`
может весить десятки мегабайт: если так, добавь в `.gitignore` строку
`landing/screencast.mp4` и заливай видео прямо в хостинг (или на YouTube и
вставь ссылку вместо `<video>`).
