# Необязательные зависимости: обложка Spotify-плейлиста из символа-экслибриса.
#
# Вынесены из requirements.txt, потому что rlPyCairo тянет pycairo, а тот
# на Linux собирается из исходников и требует системный Cairo:
#   apt-get install -y libcairo2-dev
# На Windows ставится готовым колесом, без плясок.
#
# Без этих пакетов приложение работает как обычно — плейлист просто создаётся
# со стандартной мозаикой обложек треков (services/cover_art.py это учитывает
# и мягко деградирует; проверить — rasterizer_available()).
#
# Установка:
#   pip install -r requirements-cover.txt
